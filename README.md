E-commerce frontend de produtos tecnológicos com estética feminina, desenvolvido como projeto acadêmico no curso de Análise e Desenvolvimento de Sistemas - Mackenzie.

A Artemis Glow é uma loja virtual fictícia criada para o frontend de um site de vendas online focado em tecnologia com identidade estética feminina. O projeto foi desenvolvido durante as aulas de Fundamentos Web na Universidade Presbiteriana Mackenzie, com o objetivo de aplicar na prática os conceitos de HTML e CSS abordados ao longo do curso.
O site conta com duas páginas principais: uma Home com seção sobre a marca e formulário de contato, e uma página de Produtos com catálogo organizado por categorias.

## Tecnologias Utilizadas
- HTML5;
- CSS3 com Grid Layout e Flexbox;
- Formulário com validação nativa via HTML5.

## Conceitos Aplicados
Durante o desenvolvimento do projeto, foram praticados os seguintes conceitos da disciplina:
- Semântica HTML5 - Uso de elementos header, main, section, article, figure e footer para estruturar o conteudo.
- CSS - Cosntrução de layouts utilizando Grid e Flexbox.
- Formulários HTML5 - Campos com validação nativa.
- Acessibilidade - Atributos aria-label em botôes e uso semântico de figure com figcaption.
