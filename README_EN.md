Artemis Glow is a fictional online store created as the frontend of an e-commerce website focused on technology with a feminine aesthetic identity. The project was developed during Web Fundamentals classes at Universidade Presbiteriana Mackenzie, with the goal of applying HTML and CSS concepts covered throughout the course.
The site features two main pages: a Home page with a brand overview section and a contact form, and a Products page with a catalog organized by category.

## Technologies Used
- HTML5;
- CSS3 with Grid Layout and Flexbox;
- Forms with native HTML5 validation.

## Concepts Applied
The following concepts from the course were practiced during the project's development:

- HTML5 Semantics - Use of elements header, main, section, article, figure and footer to structure the content.
- CSS - Building layouts using Grid and Flexbox.
- HTML5 Forms - Fields with native validation.
- Accessibility - Aria-label attributes on buttons and semantic use of figure with figcaption.
